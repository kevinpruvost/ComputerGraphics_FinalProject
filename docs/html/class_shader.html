Detailed Description

Contains Shader Id and methods related to the shader database. It also helps having a much lighter class because Shader_Base weighs 4 bytes and Shader only 2 bytes. And most importantly, the usage is way easier, because copies, constructors, destructors, will not affect the original shader.
