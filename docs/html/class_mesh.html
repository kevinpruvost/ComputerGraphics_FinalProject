Detailed Description

Contains Mesh Id and methods related to the mesh database. It also helps having a much lighter class because Mesh_Base weighs 30 bytes and Mesh only 2 bytes. And most importantly, the usage is way easier, because copies, constructors, destructors, will not affect the original mesh.
